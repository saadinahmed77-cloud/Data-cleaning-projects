# 🚢 Titanic Data Analysis Project

## 📌 Introduction
This project focuses on analyzing the Titanic dataset to understand the factors that influenced passenger survival. It uses data analysis and basic machine learning techniques.

## 🎯 Objectives
- To analyze the Titanic dataset  
- To identify factors affecting survival (age, gender, class)  
- To clean and preprocess the data  
- To visualize data using graphs and charts  
- To build a simple prediction model  

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  

## 📊 Dataset
The dataset contains details of passengers such as age, gender, class, fare, and survival status.

## ⚙️ Project Workflow
1. Data Collection  
2. Data Cleaning  
3. Data Analysis  
4. Data Visualization  
5. Model Building
